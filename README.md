# Logistics Management App

A full-stack logistics management system built with:

- FastAPI (Backend)
- PostgreSQL (Database)
- SQLAlchemy (ORM)
- React (Frontend - coming soon)

## Features
- Create shipments
- View shipments
- Database integration
- RESTful API

## Tech Stack
Backend:
- Python
- FastAPI
- SQLAlchemy
- PostgreSQL

Frontend:
- React (in progress)
